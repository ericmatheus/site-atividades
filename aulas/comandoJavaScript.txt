DOM = Document Object Model
DOM é um facilitador que existe dentro do navegador dentro do HTML 5 que dá acesso aos elementos diretamente pro javaScript

-Dentro do DOM temos o objeto WINDOW;
-Dentro do WINDOW temos os objetos: LOCATION, DOCUMENT, HISTORY;
-Dentro do DOCUMENT temos o objeto HTML;
-Dentro do HTML temos os objetos HEAD BODY
